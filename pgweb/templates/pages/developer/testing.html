Testing

The stability and performance of PostgreSQL is made possible by the dedicated work of people testing and reporting back their findings, whether they are bugs, security issues, or performance problems.

Below are some resources that will help you to test some of the latest code that is committed to the PostgreSQL project. These URLs contain information about an ongoing beta, the nightly development builds and documentation updates.

Any code or builds directly referenced from this section should not be considered ready for production use.
